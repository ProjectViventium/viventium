# Feature Request Workflow QA

## Scope

Owns QA for Viventium local feature-request intake, approval gates, isolated worktrees, and PR policy.

## Quality Bar

Feature-request workflows must collect success criteria before implementation, avoid the user's
primary checkout, and prevent PR creation unless artifacts and diffs are public-safe.
