## Mandatory Execution Matrix

For any production-grade claim, run and record:

- Codex host high/xhigh and Claude host max when configured and available.
- Docker Codex/Claude when the workstation image is available. If the default image fails to build
  or run, record the concrete Docker/storage/build blocker separately from host-worker results.
- Happy path, timeout/still-running, auth missing, provider rate limit, provider response failure,
  cancellation/interruption, continuation/steer, quota/concurrency, profile fallback, and stale
  worker metadata.
- File input materialization plus generated artifact preview/download/reload/continuation.
- Privacy checks for raw IDs, local paths, signed links, short refs, WebSocket/log text, DOM text,
  copy text, and public QA reports.

Every applicable case must be `PASS`, `FAIL`, `PARTIAL`, or `BLOCKED` with supporting evidence.
